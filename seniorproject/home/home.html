Are you interested in creating a well-designed and functioning site to demonstrate your strengths for networking and job recruitment? This is a complete walkthrough guide for students and incoming professionals to create a personal website portfolio that showcases their talents.



Walkthrough Process Description


  1. GitHub Configuration

   Set up the domain: create a repository and upload the starter file pack.



  2. Customization

   Edit the content: customize the HTML and add your infomation.



  3. Stylization

   Edit the design: change the CSS and make stylistic choices.



  4. Add Pages

   Expand the site: add additional pages to your liking.



  5. Tableau

   Optional: configure your site to Tableau to view data analytics in real time.




No Coding Experience Required

The walkthrough contains a downloadable starter file pack for users to open in Visual Studio Code. By following the tutorial steps, users can easily edit placeholder values in the code and learn front-end development skills on the way.



The Importance of Building a Portfolio

Showcase Your Abilities

After completing this process, users can have a site that showcases their talents. Display past educational and professional projects for colleagues and employers to conveniently access.



Develop Your Skillset

Having a website that can visualize your skills as a pre-professional is great. Creating your own website from scratch is even better and far more impressive. Increase your familiarity with HTML and CSS programming, and develop new skills in web design and layout as you complete this process.
